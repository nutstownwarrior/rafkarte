Benutzte Tools

Tools und Bibliotheken

Zur Erstellung dieser HTML-Seite, so wie dem gesamten digitalen Projektbeitrag wurden verschiedene Bibliotheken und Programme genutzt. Im Folgenden finden sie eine Liste der verwendeten Tools, und welchen Nutzen diese haben.

GitHub

GitHub ist eine Code Hosting Plattform, welche genutzt wird um zusammen Code zu schreiben. Bei unserem Projekt findet jedoch nur der Dienst GitHub pages Anwendung, welcher eine kostenlose Webseite für das Projekt bereitstellt. Den Link zu GitHub finden sie hier

Brackets

Als Entwicklungsumgebung für die Webapp wurde die Brackets IDE verwendet, diese ist unter folgendem Link zu finden: Brackets IDE

Material Design Lite

Für das allgemeine Design der HTML-Seiten wurde die Google Material Design Lite Bibliothek genutzt. Diese Bibliothek dient dazu der Seite ein ansprechendes, einheitliches Aussehen zu geben, und enthält verschiedene Komponenten, welche den Material Design Richtlinien folgen. Die offizielle Webseite der Material Design Lite Bibliothek finden sie hier: Material Design Lite

Leaflet

Die Leaflet Bibliothek wird genutzt, um die Karte sowie einige ihrer Funktionen bereitzustellen. Sie ist eine Javascript Bibliothek, welche Code enthält der das darstellen und Nutzen einer Karte ermöglichen. Leaflet

Leaflet.markercluster

Leaflet.markercluster ist ein Plugin für Leaflet, welches die Bildung von Marker Clustern hinzufügt. Diese werden erzeugt wenn sich mehrere Marker nah beieinander befinden. Den link zu dem entsprechenden Plugin finden sie hier: Leaflet.markercluster

leaflet-color-markers

Die andersfarbigen Marker Icons wurden dem GitHub Projekt leaflet-color-markers entnommen.

Mapbox

Mapbox ist ein Tool, welches die nötigen Daten zur Darstellung der Karte bereitstellt. Mapbox
